# v1.1.1 Release Notes

Date: 2026-01-27

## Highlights
- Adaptive Spaces summaries to reduce token usage
- Archived “Earlier messages (N)” toggle with full bubbles
- “Summary updated” badge after refresh

## Spaces
- Rolling summaries stored per thread
- Archived messages kept for UI only (not sent to the model)
- Live window shrinks after summary exists

## Dev Notes
- New message type: `SUMMARIZE_THREAD`
- Helper: `buildSummarizerMessages()` in `src/shared/utils.js`
- Tests: `tests/spaces-summarization.test.ts`, `tests/spaces-archive.test.ts`

- Combined OpenRouter + NagaAI model list with provider badges
- NagaAI balance support via provisioning key
- Streaming debug log toggle + download in Options
- Inline key visibility toggles for API/provisioning keys

## Additional Updates (2026-01-29)
